# 🪨 Rock Paper Scissors

A simple command-line **Rock, Paper, Scissors** game built using **Python**. Play against the computer in a first-to-*N* points match and see who wins.

---

## Features

* Play against a computer with random choices.
* Choose how many points are needed to win the match.
* Live score tracking after every round.
* Tie detection.
* Option to play multiple games without restarting the program.
* Handles invalid inputs gracefully.

---

## Technologies Used

* Python 3
* `random` module
